# My AI Tools

AIを活用して日々の業務を自動化・効率化するツール集。副業・物販・コンテンツ制作まで幅広くカバーする個人ツールリポジトリ。

## ツール一覧

| ツール | 概要 | カテゴリ |
|--------|------|----------|
| 古着出品ジェネレーター | 写真から商品タイトル・説明文を自動生成してメルカリ出品を効率化 | 物販 |
| 在庫・売上管理 | 仕入れ〜売上を一元管理し、利益率・回転率をリアルタイム把握 | 管理 |
| 勘定科目 自動振分 | 取引内容からMFクラウド用の勘定科目を自動判定 | 税務 |
| 副業法務アドバイザー | 副業・個人ビジネスの法的リスクチェックと合法的代替案を提示 | 法務 |
| AI活用プランナー | ChatGPT・Claude・画像生成AIを組み合わせた収益化プランを設計 | 戦略 |
| AIエージェントチーム | PM・法務・税務など専門ロールのAIが連携して業務を完遂 | 自動化 |

## ファイル構成

```
my-tool/
└── index.html   # AIツール紹介ランディングページ
```

## 使い方

`index.html` をブラウザで開くだけで動作します。サーバー不要。

## 技術スタック

- HTML / CSS（バニラ、依存ゼロ）
- Claude Code で開発
